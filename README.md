# 足球预测实验室：每日预测与验真

这是原“2026 世界杯 Elo 蒙特卡洛预测器”的持续验证版本。系统不再局限于杯赛，而是每天读取中国体彩网竞彩足球公开赛程，只发布北京时间未来两日、尚未开赛的足球预测，并在赛后自动验真。

## 输出内容

每场比赛固定输出五项：

1. 胜平负
2. 让球胜平负（使用体彩网当场官方让球线）
3. 比分
4. 总进球数（0—6、7+）
5. 半全场胜平负

首页同时展示累计样本数、各玩法准确率、五项全中率、历史逐场预测与真实赛果。

## 模型闭环

- 数据：体彩网竞彩足球公开赛程、胜平负/让球赔率、半场与全场赛果。
- 客观层：首次回填近 180 天官方赛果，学习球队近期进失球、滚动 Elo、联赛主客场进球、平局率、半场平局率和真实比分分布。
- 模拟层：每场固定种子运行 20,000 次带比赛节奏波动的蒙特卡洛，保留 0:0/1:1 等低比分，也保留 2:3、3:2、1:4 等开放型长尾。
- 自洽决策：先判断胜/平/负方向，再在这个方向内选择代表比分；让球胜平负和总进球由该比分直接推导，半全场的全场方向也必须一致。
- 平局建模：市场平赔、联赛平局率和半场平局率分别进入全场与半场判断，页面同时展示平局概率及半场平候选。
- 主观层：`data/adjustments.json` 可录入已经核验的伤停、轮换、天气、动机修正；必须写明理由和失效时间。没有可靠信息时不做臆测。
- 学习：完赛后根据赛前已冻结样本的 Brier 分数、进球残差和主场残差，收缩更新市场权重、总进球尺度和主场修正。

预测写入 `data/history.json` 后即冻结。模型升级不会回算或改写旧预测，因此历史准确率是可审计的真实前瞻成绩，不是赛后回放拟合成绩。

## 本地运行

需要 Node.js 20 或更高版本，无第三方依赖。

```bash
npm test
npm run update
npm run serve
```

浏览器打开 `http://127.0.0.1:4173`。Windows 也可以直接双击 `启动本地预览.bat`；如需先联网更新，双击 `立即联网更新.bat`。

## 自动更新

`.github/workflows/update-predictions.yml` 每 30 分钟运行一次：

1. 拉取未来赛程与近 45 天赛果；
2. 结算已冻结预测；
3. 更新 Elo 和校准参数；
4. 仅为未开赛且未发布过的比赛生成预测；
5. 提交 `data/*.json`，GitHub Pages 随之更新。

若仓库启用了分支式 GitHub Pages，网页入口仍是：

`https://xkxjj.github.io/daily-football-predictor/`

## 重要说明

准确率随玩法难度差异很大，尤其“精确比分”天然是低命中任务；界面不会把不同玩法混成一个看似很高的数字。任何预测都不能保证正确或盈利，本项目只用于模型研究、公开验证和复盘，不构成投注建议。
