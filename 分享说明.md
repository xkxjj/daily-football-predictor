# 2026 世界杯预测器分享说明

## 离线发文件

把 `index.html` 发给朋友即可。朋友下载后双击打开，就能在自己的电脑上使用。

注意：这种方式是固定版本。朋友已经下载的文件不会跟着你这边更新。

## 同一 Wi-Fi / 局域网分享

双击 `双击启动局域网分享.bat`。

窗口里会显示类似 `http://192.168.x.x:8766/` 的网址。把这个网址发给同一 Wi-Fi 或同一局域网内的朋友。

注意：`127.0.0.1` 只代表当前这台电脑，不能直接发给朋友用。

## 公开链接同步分享

把整个文件夹上传到静态网页托管服务即可。推荐 GitHub Pages、Cloudflare Pages、Netlify 或 Vercel。

朋友访问同一个公开链接时，页面每次打开都会自动拉取线上赛果；页面保持打开时，每 5 分钟也会自动同步一次。

如果线上赛果接口暂时不可用，页面会使用本地内置赛果兜底，不影响打开。
